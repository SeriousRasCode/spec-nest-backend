# User Management API (NestJS)

A simple backend API built with NestJS, Prisma, and SQLite.
It demonstrates clean module-based architecture and CRUD operations.
Users can be created, retrieved, updated, and deleted via REST endpoints.
Prisma is used as the ORM for database access.
SQLite is used for simplicity during development.
DTOs and class-validator handle request validation.
The project follows Controllers → Services → Database flow.
It is structured to be easily extendable with authentication.
Run `npm install` to install dependencies.
Start the server with `npm run start:dev`.
